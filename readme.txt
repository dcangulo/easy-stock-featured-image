Automatically attach stock photos as featured image to your posts without featured image.

== Description ==
<strong>Easy Stock Featured Image</strong> automatically attach stock photos as featured image to your posts without featured image.

Just install the plugin and the set your preference. No coding skills needed.

= How does it work? =
Checks if the post (any post type with thumbnail support, including pages) has already a featured image associated, if it doesn't have one, it will fetch a stock photo from Unsplash and attach it as featured image.

= Requirements =
* WordPress 5.4 or higher.
* PHP 7.2 or higher.

== Installation ==
1. In your WordPress admin menu on the left hover to Plugins and click Add New.
2. Enter "Easy Stock Featured Image" in the text box in the upper right corner.
3. Find the plugin entitled "Easy Stock Featured Image" by "David Angulo" then click Install Now.
4. Click Activate once the plugin is installed.
5. Go to the plugin settings and configure the settings then save.
6. View a post without featured image and it should have one.
